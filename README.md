# BO3-Custom-Maps-Installer

BO3-Custom-Maps-Installer is a tool that facilitates the installation of custom maps on the Non-Steam version of Call of Duty: Black Ops III.

## Features
* Drag and drop map installation
* Remove useless files to save space
* Rename the map to match the workshop map name
* Easy to use
